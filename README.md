# ACIM Controller

Advanced Controller Input Manager for Windows.

ACIM Controller allows users to manage physical and virtual game controllers with precision, featuring built-in support for HidHide driver integration.

## Features
- Hide physical controllers from specific applications.
- Low-latency input processing.
- Professional-grade UI for easy configuration.

## Installation
1. Download the latest `ACIM-Controller-Setup.exe` from the **Releases** section.
2. Run the installer as Administrator.
3. Follow the on-screen instructions to install the required HidHide drivers.
